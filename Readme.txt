This Readme.txt file gives an overview of the directory and file structure.
The following directories and files are in each thesis directory.

bin                   Any executables and scripts you developed or used
Paper                 Scientific publications related to this thesis; these
                      have to be named as follows: YYYY-Title.pdf
Presentations         Slides used for your presentations
Results               Results/data of your measurements and experiments
Src                   Source code developed by the student
Text                  Thesis written in LaTeX
 |- text              Individual parts (sections and chapters) of the thesis
 |- graphics          Graphics used in the thesis
 |- bibliography.bib  Bibtex file of all read papers
Thesis-Management     Description and management of the thesis
Readme.txt            This file
